# psadmin-io-util
Utilities for PeopleSoft from the psadmin.io community.

# Deploy
Quick example showing how to deploy these utilites

```
# install git
# setup env vars
# clone repo
cd %PS_HOME_CUST%\sdk
git clone https://github.com/psadmin-io/psadmin-io-util.git 
# compile
## set weblogic env
## add classpath
## compile 
## create elf jar
# setup weblogic
## filter in web.xml
## elf logging
# bounce
```
